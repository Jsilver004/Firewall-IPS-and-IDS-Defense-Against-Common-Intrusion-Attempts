# Firewall-IPS-and-IDS-Defense-Against-Common-Intrusion-Attempts

## Research Question

How effective are Firewall, Intrusion Prevention Systems (IPS) and Intrusion Detection Systems (IDS) technologies at detecting and preventing common intrusion attempts in a controlled network environment?

## Objective

Compare the detection and prevention capabilities of firewall, IPS and IDP technologies against common network attacks.
